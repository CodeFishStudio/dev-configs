# CodeFish Studio Dev Configs

Shared configurations for dev tools across CodeFish Studio projects.

## 📦 Installation

```bash
bun add -d @codefish/dev-configs@github:CodeFishStudio/dev-configs#v0.1.13
```

New projects are set up automatically via [create-cfs-app](https://github.com/CodeFishStudio/create-cfs-app).

## 💪 Manual Setup

Refer to the [Manual Setup](./docs/MANUAL_SETUP.md) instructions.
